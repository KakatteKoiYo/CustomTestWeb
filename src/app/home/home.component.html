¿Qué es CustomTestWebsite?

CustomTestWebsite sirve para realizar test personalizados.

Creas tus propias listas, las cargas y la página automáticamente genera un test con preguntas y cinco opciones aleatorias. La respuesta es colocada en una posición aleatoria también. El formato en el que se crean las listas es el siguiente: palabra1 = palabra2.

Tu decides si la pregunta será la palabra1 o palabra2. Si decides que la pregunta serán las palabra1, entonces las posibles respuestas serán las palabra2 Si aun no tienes ninguna lista puedes probar crear una: CREAR LISTA

O si lo prefieres puedes probar utlizando la lista prehecha que contiene un Kanji japonés con su significado y estudiar dicha lista seleccionada acá: ESTUDIAR LISTA
